? True Positives: Correct positive predictions.
? True Negatives: Correct negative predictions.
? False Positives: Incorrect positive predictions.
? False Negatives: Incorrect negative predictions.

Accuracy ?Accuracy: Proportion of correct predictions (TP + TN) / (TP + TN + FP + FN).

Precision ?Precision: Proportion of correct positive predictions TP / (TP + FP).

Recall ?Recall: Proportion of actual positives correctly identified TP / (TP + FN).

Specificity ?Specificity: Proportion of actual negatives correctly identified TN / (TN + FP).

NPV ?NPV: Proportion of predicted negatives that are correct TN / (TN + FN).

F1 Score ?F1 Score: Harmonic mean of Precision and Recall 2 * (Precision * Recall) / (Precision + Recall).
